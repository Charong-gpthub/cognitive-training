## 二、理论背景

### 2.1 历史起源

心理旋转任务由罗杰·谢泼德（Roger Shepard）和梅茨勒（Metzler）于1971年首次提出。该任务是研究空间认知和心理旋转能力的经典实验范式。

### 2.2 理论基础

**核心概念**：
- **心理旋转**：在脑海中旋转物体的能力
- **空间表征**：内部空间信息的表示
- **旋转角度效应**：旋转角度越大，反应时间越长

## 三、游戏概述

### 3.1 任务描述

屏幕显示一个参考图形和若干选项，要求判断哪些选项是参考图形的旋转版本（可能镜像）。

### 3.2 应用场景

| 场景 | 说明 |
|------|------|
| 空间能力评估 | 测量心理旋转能力 |
| 认知训练 | 提升空间想象力 |
| 职业选拔 | 航空、工程等领域 |
