## 二、理论背景

### 2.1 历史起源

数字广度测试是韦氏智力量表（Wechsler Adult Intelligence Scale, WAIS）的核心子测验之一，最早由大卫·韦克斯勒（David Wechsler）于1955年开发。该测试用于评估个体的短时记忆容量。

### 2.2 理论基础

**核心概念**：
- **短时记忆容量**：Gatherer于1956年提出短时记忆容量约为7±2个组块
- **语音环路**：Baddeley工作记忆模型中的语音信息存储系统
- **组块化**：将信息组织成有意义单元的策略

## 三、游戏概述

### 3.1 任务描述

屏幕上快速依次显示一串数字，数字显示完毕后，玩家需要按显示顺序（正向）或相反顺序（逆向）输入这些数字。

### 3.2 应用场景

| 场景 | 说明 |
|------|------|
| 认知评估 | 测量短时记忆容量 |
| 认知训练 | 提升数字记忆能力 |
| 临床评估 | 评估记忆障碍 |

## 十、技术实现要点

### 10.1 核心算法

```javascript
function generateDigits(length) {
    let digits = '';
    for(let i = 0; i < length; i++) {
        digits += Math.floor(Math.random() * 10);
    }
    return digits;
}
```
