## 二、理论背景

### 2.1 历史起源

持续表现任务（Continuous Performance Test, CPT）最早由Rosvold等人于1956年开发，用于评估持续注意力和冲动控制。该任务被广泛用于ADHD（注意缺陷多动障碍）的诊断和认知功能评估。

### 2.2 理论基础

**核心概念**：
- **持续性注意**：长时间维持注意力在单调任务上的能力
- **警觉性**：对意外刺激保持敏感的能力
- **冲动控制**：抑制不适当反应的能力

---

## 三、游戏概述

### 3.1 任务描述

屏幕上快速连续呈现一系列刺激（通常是字母或数字），要求被试在看到特定目标时做出反应，而对非目标刺激不做反应。该任务评估持续注意力和冲动控制。

### 3.2 应用场景

| 场景 | 说明 |
|------|------|
| ADHD诊断 | 评估注意力和冲动控制 |
| 认知训练 | 提升持续注意力 |
| 临床评估 | 评估警觉性 |
