## 二、理论背景

### 2.1 历史起源

舒尔特方格（Schulte Grid）是由德国精神科医生弗朗茨·舒尔特（Franz Schulte）于1971年提出的一种注意力训练工具。该训练方法最初用于评估和治疗注意力缺陷问题，后被广泛应用于飞行员选拔和认知训练领域。

### 2.2 理论基础

**核心概念：视觉搜索与注意稳定性**

舒尔特方格通过要求被试在最短时间内按顺序找出所有数字，训练视觉扫描速度和注意力稳定性。

**理论解释**：
- **注意广度**：一次性能够注意到的信息量
- **视觉搜索效率**：在复杂视觉环境中快速定位目标的能力
- **注意稳定性**：在持续任务中保持注意力的能力

### 2.3 神经机制

完成舒尔特方格任务时，主要激活以下脑区：

| 脑区 | 功能 | 在任务中的作用 |
|------|------|----------------|
| **枕叶皮层** | 视觉加工 | 视觉信息处理 |
| **顶叶** | 空间注意 | 视觉搜索引导 |
| **前额叶** | 执行控制 | 注意力调控 |

## 三、游戏概述

### 3.1 任务描述

屏幕上显示一个N×N的数字方格（通常为5×5），方格中随机分布着1到N²的数字。玩家需要按照1、2、3...的顺序依次点击所有数字，记录完成时间。

### 3.2 应用场景

| 场景 | 说明 |
|------|------|
| **认知训练** | 提升视觉搜索速度和注意力稳定性 |
| **飞行员选拔** | 用于评估注意力和反应速度 |
| **儿童教育** | 训练儿童注意力和数学基础 |
| **康复训练** | 用于注意力缺陷康复 |

### 3.3 目标人群

| 人群 | 适用性 | 注意事项 |
|------|--------|----------|
| 儿童（6岁以上） | 适用 | 从较小网格开始 |
| 青少年 | 适用 | 标准难度 |
| 成年人 | 适用 | 可挑战更大网格 |
| 老年人 | 适用 | 评估注意衰退 |

## 五、游戏规则

### 5.1 核心规则

- 必须按递增顺序点击数字（1→2→3→...→N²）
- 点击错误不暂停，但会影响得分
- 记录从点击第一个数字到完成最后一个数字的时间

### 5.2 约束条件

| 条件 | 说明 |
|------|------|
| 时间限制 | 无硬性时间限制 |
| 错误处理 | 错误点击不暂停，但记录错误次数 |
| 放弃 | 可手动结束，保留当前成绩 |

## 十、技术实现要点

### 10.1 核心算法

```javascript
function generateGrid(size) {
    const total = size * size;
    const numbers = Array.from({length: total}, (_, i) => i + 1);
    // 洗牌算法
    for (let i = numbers.length - 1; i > 0; i--) {
        const j = Math.floor(Math.random() * (i + 1));
        [numbers[i], numbers[j]] = [numbers[j], numbers[i]];
    }
    return numbers;
}
```
