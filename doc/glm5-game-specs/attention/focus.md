## 二、理论背景

### 2.1 历史起源

中科院注意力训练（CAS - Attention）是中国科学院心理研究所开发的注意力评估与训练系统。该系统基于注意力的多成分理论，结合了视觉广度测量和注意分配训练。

### 2.2 理论基础

**核心概念：视觉广度与注意分配**

- **视觉广度**：一次性视觉扫描所覆盖的范围
- **注意分配**：同时关注多个目标的能力
- **视觉搜索效率**：在复杂场景中快速定位目标

### 2.3 神经机制

| 脑区 | 功能 |
|------|------|
| 顶叶 | 空间注意 |
| 枕叶 | 视觉加工 |
| 前额叶 | 注意力调控 |

## 三、游戏概述

### 3.1 任务描述

屏幕上显示不规则分布的数字碎片，玩家需要在碎片中快速寻找目标数字。该训练旨在打破视觉定势，扩展视幅广度。

### 3.2 应用场景

| 场景 | 说明 |
|------|------|
| 认知训练 | 提升视觉广度和搜索效率 |
| 注意力评估 | 测量视觉注意范围 |
| 儿童教育 | 训练视觉分辨能力 |
