## 一、基础信息

| 属性 | 内容 |
|------|------|
| 中文名称 | 认知能力综合测试 |
| 英文名称 | Cognitive Assessment Battery |

## 二、功能说明

综合评估多项认知能力，包括注意力、工作记忆、推理能力等。

---

## 三、评分体系

| 指标名称 | 说明 |
|----------|------|
| 综合得分 | 各维度加权总分 |
| 各维度得分 | 注意力、记忆、推理等分项 |
