## 一、基础信息

| 属性 | 内容 |
|------|------|
| 中文名称 | 每日训练报告 |
| 英文名称 | Daily Report |

## 二、功能说明

展示用户的训练历史数据，包括各项认知能力的训练记录和进步趋势。

---

## 三、主要功能

- 训练历史记录查询
- 成绩趋势图表
- 能力雷达图
- 训练建议
