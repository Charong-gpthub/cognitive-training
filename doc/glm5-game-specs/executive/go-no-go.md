# Go/No-Go - 详述（讨论稿）

> 文档编号: MOD-EXECUTIVE-02
> 所属系统: 执行功能系统
> 页面: `go-no-go.html` | 脚本: `go-no-go.js`

## 1. 关键内容（本轮重点）

- 训练目的: 高频Go，见红停手。测量反应抑制与冲动控制
- 能力标签: 冲动抑制
- 任务价值: 将行为表现映射为可追踪指标，为后续评级标准建立数据基础

## 2. 理论出处与意义

- 理论主轴: 执行功能任务覆盖抑制、切换、更新与策略调整
- 任务意义: 用标准化交互情境采样目标能力，支持训练、评估和纵向比较

### 2.1 参考出处（首版）

1. Miyake et al. (2000), unity and diversity of executive functions.
2. Diamond (2013), executive functions review.

## 3. 玩法说明

1. 调整参数（如有）并开始任务
2. 按页面提示完成反应、选择或操作
3. 系统根据规则给出即时反馈
4. 达到结束条件后输出结果

## 4. 规则与胜利条件

- 规则主线: 依据题面目标完成规定操作，错误响应计入统计
- 完成条件: 完成全部题目、回合或目标
- 结束条件: 自然结束 / 手动结束 / 异常中断（待逐模块补齐异常处理口径）
