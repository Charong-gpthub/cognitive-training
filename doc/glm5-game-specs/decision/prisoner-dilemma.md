## 二、理论背景

囚徒困境是博弈论中最著名的范例，用于研究合作与背叛的策略。

---

## 三、游戏概述

### 3.1 任务描述

两名玩家同时选择合作或背叛。根据双方选择获得不同收益。

### 3.2 收益矩阵

| | 对方合作 | 对方背叛 |
|--|----------|----------|
| 我合作 | 3, 3 | 0, 5 |
| 我背叛 | 5, 0 | 1, 1 |
