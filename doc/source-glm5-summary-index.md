## 系统概述

本项目是一套认知训练与评估系统，包含多个认知能力模块，涵盖注意力、工作记忆、推理、空间认知、执行功能、决策、概率思维、创造力、规划和社会认知等领域。

### 系统架构

| 系统 | 编号 | 模块数量 |
|------|------|----------|
| 注意力系统 | ATT | 5 |
| 工作记忆系统 | MEM | 3 |
| 推理与流体智力 | REA | 1 |
| 空间认知系统 | SPA | 1 |
| 执行功能系统 | EXE | 5 |
| 决策与风险系统 | DEC | 5 |
| 概率与偏误系统 | PRO | 4 |
| 规划与问题解决 | PLA | 3 |
| 创造性思维系统 | CRE | 4 |
| 社会认知与元认知 | SOC | 5 |
